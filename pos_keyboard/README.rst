Module allows to use usual keyboard (not virtual one) in Point of Sale.
================================================================
Usage:

=========== ===================== =================
Type        Numpad                Extra keys  
=========== ===================== =================
mode qty    ``/``                 ``q``
----------- --------------------- -----------------
mode disk   ``-``                 ``d``
----------- --------------------- -----------------
mode price  ``*``                 ``p``
----------- --------------------- -----------------
+/-         ``+``                 ``s``
=========== ===================== =================

Tested on Odoo 8.0 9a5b6c8b98d4db3a190c3ac46bd5f3a6df3069b2
